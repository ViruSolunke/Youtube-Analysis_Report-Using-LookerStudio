## 🚀 Project Overview

Understanding YouTube performance goes beyond counting views.

This dashboard provides a complete analytical view of channel growth, viewer engagement, content effectiveness, and audience interaction. By combining multiple performance indicators into a single visual interface, creators can make smarter content decisions backed by data.

---

## 🎯 Business Questions Answered

🔹 Which videos generate the highest watch time?

🔹 What content drives maximum audience engagement?

🔹 How do likes, comments, and shares influence performance?

🔹 Which videos attract the most subscribers?

🔹 What trends can be identified from viewer behavior?

🔹 Which content strategy delivers the best results?

---

## 📈 Key Performance Indicators (KPIs)

| Metric                   | Description                  |

| 👀 Views                 | Total video views generated  |
| ⏱️ Watch Hours           | Total viewing time           |
| 📺 Average View Duration | Viewer retention indicator   |
| 👍 Likes                 | Positive audience engagement |
| 💬 Comments              | Community interaction        |
| 🔄 Shares                | Content virality measurement |
| 🔔 Subscribers           | Channel growth tracking      |
| 👎 Dislikes              | Negative feedback analysis   |
